    class ParsedResult(BaseModel):

    3. Create `services/stake/results/evaluator.py` with function `evaluate_bets(final_bets: list[dict], result: ParsedResult) -> list[BetOutcome]`:
    - For each bet in final_bets:
      - Extract runner_number, bet_type, usdt_amount, decimal_odds, place_odds from the bet dict
      - winner = result.finishing_order[0] if result.finishing_order else None
      - top_3 = set(result.finishing_order[:3])
      - If bet_type == "win":
        - won = (winner == runner_number)
        - payout = amount * decimal_odds if won else 0.0
        - profit = payout - amount
        - evaluable = True (win bet always evaluable if winner known)
        - If winner is None: evaluable = False, profit = 0.0
      - If bet_type == "place":
        - If result.is_partial: evaluable = False, won = False, profit = 0.0
        - Else: won = (runner_number in top_3), payout = amount * place_odds if won else 0.0, profit = payout - amount
        - If place_odds is None: evaluable = False
      - Round profit to 4 decimal places
      - Return BetOutcome with all fields populated
    - ARCH-01: This is pure Python. No LLM calls.
