**Add `sizing_node` to `services/stake/pipeline/nodes.py`** (sync function, pure Python per ARCH-01):
- Takes `PipelineState`, returns partial update dict
- If `state.get("skip_signal")` is True, return `{}`
- Gets `analysis_result` dict from state
- If `analysis_result["overall_skip"]` or `analysis_result["ai_override"]`: return `{"skip_signal": True, "skip_reason": analysis_result.get("skip_reason") or analysis_result.get("override_reason"), "skip_tier": 2, "final_bets": []}`
- Gets bankroll from `BankrollRepository(settings.database_path).get_balance()` (or state if available)
- Gets sizing settings from `get_stake_settings().sizing`
- For each recommendation in `analysis_result["recommendations"]`:
  - Skip if `label == "no_bet"`
  - Compute: `ev_val = expected_value(rec["ai_win_prob"], runner_decimal_odds)` (look up decimal_odds from enriched_runners by runner_number)
  - If ev_val <= 0: skip (negative EV)
  - Compute: `kf = kelly_fraction(rec["ai_win_prob"], runner_decimal_odds)`
  - Compute: `amount = bet_size_usdt(bankroll, kf, settings.sizing.kelly_multiplier, settings.sizing.per_bet_cap_pct, settings.sizing.min_bet_usdt)`
  - Determine `bet_type`: "win" if label in ("highest_win_probability", "best_value"), "place" if "best_place_candidate"
  - For place bets: use `place_bet_ev(rec.get("ai_place_prob") or rec["ai_win_prob"] * 0.6, place_odds)` to check if place bet is +EV. Look up place_odds from enriched_runners. If place EV negative, skip.
  - Check `data_sparse` from research_results for this runner; if sparse, apply: `amount = apply_sparsity_discount(amount, True, settings.sizing.sparsity_discount)`
  - Build bet dict: `{"runner_name": ..., "runner_number": ..., "label": ..., "bet_type": ..., "ev": ev_val, "kelly_pct": round(kf * 100, 2), "usdt_amount": amount, "data_sparse": ..., "reasoning": rec["reasoning"]}`
- Sort by EV descending
- Apply `apply_portfolio_caps(raw_bets, bankroll, settings.sizing.max_total_exposure_pct, settings.sizing.max_win_bets)`
- Return `{"final_bets": final_bets}`
