  <behavior>
    ## Math Functions (services/stake/parser/math.py)

    ### no_vig_probability(implied_prob: float, book_overround: float) -> float
    - no_vig_probability(0.5, 1.15) returns approximately 0.434783 (0.5 / 1.15)
    - no_vig_probability(0.25, 1.10) returns approximately 0.227273
    - Result always <= implied_prob (removing margin shrinks probability)

    ### expected_value(ai_prob: float, decimal_odds: float) -> float
    - expected_value(0.5, 3.0) returns 0.5 (positive EV: 0.5*2 - 0.5)
    - expected_value(0.3, 2.5) returns -0.25 (negative EV: 0.3*1.5 - 0.7)
    - expected_value(0.0, 5.0) returns -1.0 (zero prob = lose entire stake)

    ### kelly_fraction(ai_prob: float, decimal_odds: float) -> float
    - kelly_fraction(0.5, 3.0) returns 0.25 (full Kelly for 50% at 3.0)
    - kelly_fraction(0.3, 2.5) returns 0.0 (negative EV = no bet)
    - kelly_fraction(0.6, 2.0) returns 0.2 (0.6*1 - 0.4 / 1)
    - Never returns negative values

    ### bet_size_usdt(bankroll, kelly_f, kelly_multiplier=0.25, per_bet_cap_pct=0.03, min_bet=1.0) -> float
    - bet_size_usdt(1000, 0.10) returns 25.0 (1000 * 0.10 * 0.25)
    - bet_size_usdt(1000, 0.20) returns 30.0 (1000 * 0.20 * 0.25 = 50, capped at 3% = 30)
    - bet_size_usdt(100, 0.01) returns 0.0 (100 * 0.01 * 0.25 = 0.25, below min_bet 1.0)
    - bet_size_usdt(1000, 0.10, kelly_multiplier=1.0) returns 30.0 (capped at 3%)

    ### apply_portfolio_caps(bets, bankroll, max_total_pct=0.05, max_win_bets=2) -> list[dict]
    - 3 win bets: only first 2 by EV are kept (max_win_bets=2)
    - Total capped at 5% of bankroll; excess bets trimmed
    - Bets below 1.0 USDT after trimming are dropped
    - Place bets not counted against max_win_bets
    - Returns new list (no mutation of input)

    ### apply_sparsity_discount(amount: float, data_sparse: bool, discount: float = 0.5) -> float
    - apply_sparsity_discount(10.0, True) returns 5.0
    - apply_sparsity_discount(10.0, False) returns 10.0
    - apply_sparsity_discount(1.5, True) returns 0.0 (0.75 below min_bet 1.0)

    ### place_bet_ev(ai_place_prob: float, place_odds: float) -> float
    - place_bet_ev(0.7, 2.0) returns 0.4 (0.7*1.0 - 0.3)
    - place_bet_ev(0.3, 1.5) returns -0.55 (0.3*0.5 - 0.7)
    - Uses place_odds, NOT win_odds (BET-07)

    ## Pre-Skip Check Node (services/stake/pipeline/nodes.py)

    ### pre_skip_check_node(state: PipelineState) -> dict
    - State with overround_active=1.20 (margin 20%) and threshold 15.0 returns {"skip_signal": True, "skip_reason": "...", "skip_tier": 1}
    - State with overround_active=1.10 (margin 10%) returns {"skip_signal": False}
    - State with overround_active=None returns {} (no skip decision)
  </behavior>
