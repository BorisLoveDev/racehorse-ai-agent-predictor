   - Docstring: `"Search for horse racing information using an AI model with web access. Use for: runner form, trainer stats, expert opinions."`

   - Takes `StakeSettings` as parameter
   - If `settings.research.provider == "online"`: use `ChatOpenAI` with `openai_api_base="https://openrouter.ai/api/v1"`, `openai_api_key=settings.openrouter_api_key`, `model=settings.research.model`, `temperature=settings.research.temperature`, `max_tokens=settings.research.max_tokens`, `extra_body={"plugins": [{"id": "web"}]}`.    - Creates agent via `create_react_agent(model=llm, tools=tools, response_format=ResearchOutput, prompt=RESEARCH_SYSTEM_PROMPT)`
   - Returns the compiled agent

2. **`research_node(state: PipelineState) -> dict`** async function:
   - Per D-06: if `state.get("skip_signal")` is True, return `{}` immediately (skip already decided)
   - Gets settings via `get_stake_settings()`
   - Builds research agent via `build_research_agent(settings)`
   - Constructs runner context from `state["enriched_runners"]` and `state["parsed_race"]`: for each active runner, include name, number, jockey, trainer, form_string, decimal_odds, track, race_number, distance, surface
   - Invokes agent: `result = await agent.ainvoke({"messages": [HumanMessage(content=runners_context)]})`
   - Extracts structured response: `research_output = result.get("structured_response")`
   - Returns `{"research_results": research_output.model_dump() if research_output else None, "research_error": None}`
   - On exception: returns `{"research_results": None, "research_error": str(e)}`

   - Format: clear sections so the agent can parse it
